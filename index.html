What Is HTML



HTML:
H - Hyper
T - Text
M - Markup
L - Language
HTML is used as thet structure of websites, and what makes it special is that you can link to multiple html documents, making it easier to share text online. While the intent of the original creators wasn't to create a stylish way to display information, as the demand for internet grew larger, using simple styling tags such as the i tag wasn't going to work, and that is where the story of css started.

What Is HTML



CSS:
C - Cascading
S - Style
S - Sheets
CSS was a revolutionary invention for the internet, because it allowed you to seperate your styling from your structure, and it made doing advanced styling a million times easier. The introduction of CSS to the web changed our expectations of the internet, and you could start to see many companies use this new invention on their websites, but there is still something missing, interactivity, and that is when JavaScript is introduced.

What Is JavaScript



Now one thing that confuses many people is the name, while it has the word Java in its name, it is no where near Java. JavaScript is a very interesting language right now, because at the start, JavaScript was used in websites, such as this one, for interactivity, but then google introduced the V8 engine which allowed JavaScript to be used on servers as a backend, but we'll be talking about the web based JavaScript. This JavaScript allows you to traverse the DOM (DocumentObjectModel) and change the webpage dynamically, without HTML or CSS. This was crazy, because before JavaScript, you could only build your web pages to a certain point of beauty, but now you can animate the page way easier, and you can now make 3d apps, browser games, 3d animations, and more that you couldn't really do before JavaScript.
